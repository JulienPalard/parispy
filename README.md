# ParisPy Meetup Website

This is hosted on [parispy.org](https://www.parispy.org).

## Local Setup

```sh
pip3 install -r requirements.txt
```

## Local run

```sh
pelican --autoreload --listen
```

then navigate to [http://localhost:8000](http://localhost:8000)
